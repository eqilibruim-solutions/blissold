Web Push Notifications

Using OneSignal

Create OneSignal Account

This module allows to sends personalized push notifications to subscribed users from websites.

After successfully creating OneSignal account create new app for website push platforms.

Supported Web Browsers

Only works for secure websites (with HTTPS://)

  • Google Chrome (By Default)
  • Mozilla Firefox (By Default)
  • Apple Safari (Optional)

Configuration in Odoo

Website Admin > Configuration > OneSignal Web Push Notifications

After Successful Configuration

You can see dialog in your website that ask users for enable push notifications.
